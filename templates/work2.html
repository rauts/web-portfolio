Regression Model On eCommerce Data

In this project, I adviced an eCommerce company whether they put more effort on improving user experience on mobile app or desktop. For this task, I applied multivariate regression model on customer data to find the answer. Please click on GitHub icon below to view the project.

Logistic Regression Model On Advertising Data

This project answers the question whether or not a particular internet user clicked on an Advertisement on a company website. To answer this question, I used a logistic regression model on an addertising dataset. Please click on GitHub icon below to view the project.

As an investor, you would want to invest in people who showed a profile of having a high probability of paying you back. In this project, I created a model that would help predict this in that I used decision tree and random forest models on a loan dataset. Please click on GitHub icon below to view the project.

Given the properties of an iris flower, the model should predict its type. This is a classification problem, and I created a model that would help predict this for which I used a support vector machine model on an iris dataset. Please click on GitHub icon to view the project.
